# JetUse on OCI

JetUse（ジェットユース）は、OCI の Data & AI / 生成AIサービスを使って、チャット、RAG、DBチャット、議事録、翻訳・要約、エージェントなどをまとめて試せる実装テンプレートです。

## 公開サイト

https://oracle-japan.github.io/jetuse/

## 主な機能

- OCI Generative AI を使ったチャット
- RAG による社内データ・ドキュメント検索
- DB Chat / NL2SQL
- 議事録、翻訳、要約、映像分析
- エージェント機能
- カスタムエージェント・ユースケースの作成

## 対象ユーザー

- OCI で生成AIを試したい開発者・技術検証担当者
- RAG やエージェントの PoC を始めたいチーム
- OCI Data & AI サービスの構成例を確認したい人
